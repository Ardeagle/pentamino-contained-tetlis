# TETRIS + PENTOMINO

本作は、従来のテトリス（7種のテトロミノ）に、5ブロックで構成される「ペントミノ」18種を統合した拡張型パズルゲームです。Frutiger Aero調の透明感のあるUIとブロックデザインを採用し、動的なペア・バッグ生成システムと固有のスコア倍率仕様によって高度な盤面管理を要求します。

## 1. 基本操作 (Controls)

* **← / →** : ミノの左右移動
* **↓** : ソフトドロップ
* **↑** : ハードドロップ
* **X** : 右回転 (時計回り)
* **Z** : 左回転 (反時計回り)
* **Space** : ホールド (1回のみ、設置後にリセット)
* **SHIFT+B** : ボム発動 (上部10段を面制圧 / 最大3回)
* **Esc** : メニューへ戻る
* **マウス左クリック** : (プラクティスモード専用) 盤面へのブロック直接配置/消去

## 2. ゲームモード (Modes)

### NORMAL GAME

144ライン消去を目標とするスコアアタックモードです。
レベル（落下速度）は12段階で上昇し、ゲームオーバーまたは144ライン到達時にリザルト画面に詳細な統計（各種スピン回数、BtoB、MAX RENなど）が表示されます。

### PRACTICE MODE

ブロックの自由配置や、落下状態（重力）のON/OFF切り替えが可能な検証モードです。
特定のペントミノを任意に呼び出せる「ビジュアル召喚パレット」を使用し、壁蹴り（SRS）の挙動やスピンのセットアップをテストできます。

## 4. 出現アルゴリズム (Spawn Systems)

ゲーム開始前の「OPTION」にて、ペントミノの混入数（1, 2, 4, 6, 8, 18）および出現方式を選択できます。

* **Random Mix (ランダム方式)**
* 有効化されているクラスの中から、指定された個数のペントミノが完全にランダムで選出され、基本の7-BAGに混入されます。


* **Pair System (ペア・BAG巡回方式)**
* ミノを単体ではなく「鏡像」や「特性の近いもの」同士の **ペア** として管理します。
* バッグ内にペア単位で投入されるため、「Iが出たなら同じサイクル内で確実にTが出る」「Z5が出たなら確実にS5が出る」という予測可能性が担保されます。
* 未選択のクラスに属するペアは自動的に除外されます。



**【ペアテーブル】**

* Alternative: `[I5, T5]`, `[Z5, S5]`, `[L, J5]`
* Complex: `[P, G]`, `[N, R]`
* Distoted: `[Y, B]`, `[F, E]`
* Exotic: `[X, U]`, `[W, V]`

## 5. 特殊システム (Special Mechanics)

* **BOMB (ボム機能)**
* 盤面が危険な状態に陥った際、`B`キーで発動。盤面上部を一掃し、下部に届く穴をあけます。
* 発動時は専用の面爆発エフェクト（閃光とパーティクル）が発生します。
* **クリア時ボーナス**: 144ライン到達時にボムが残っている場合、1個につき **15,000点** が加算されます。1度も使用せず3つ残した場合は、Perfect Bonusとして追加で **50,000点** が付与されます。


* **PENTRIS (ペントリス)**
* I_penta（5マス直線）などを使用し、一度に5ラインを同時消去する技術です。TETRIS（4ライン）を凌ぐ高スコアを獲得できます。


* **図鑑機能 (Library)**
* オプション画面から「クラス解説＆形状図鑑」を開くことができます。現在の表示名称（カスタム名）やペアの構成を視覚的に確認可能です。
